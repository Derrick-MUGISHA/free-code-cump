Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages worldwide.

It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

print("Hello, Python!") import this # The Zen of Python
Basic Syntax

Python uses indentation to define code blocks, which is different from many other programming languages.

Statements are typically written on a single line, and the colon (:) is used to start code blocks.

  • Indentation is crucial in Python
  • Use 4 spaces for each indentation level
  • Consistent indentation prevents syntax errors
if True: print("This is an indented block")
Data Types

Python has several built-in data types to store different kinds of information.

The most common data types include integers, floating-point numbers, strings, lists, and dictionaries.

  • Integers: Whole numbers like 42
  • Floats: Decimal numbers like 3.14
  • Strings: Text enclosed in quotes
  • Lists: Ordered, mutable collections
  • Dictionaries: Key-value pair collections
age = 30 # Integer name = "Python Learner" # String
Functions

Functions in Python are defined using the 'def' keyword and can take parameters and return values.

They help in organizing code and promoting reusability.

def greet(name): return f"Hello, {name}!"
  • Use 'def' to define functions
  • Function names should be lowercase
  • Use descriptive function names
Modules

Modules are Python files containing functions, classes, and variables that can be imported and used in other scripts.

Python comes with a rich standard library of modules for various tasks.

import math # Importing a standard library module from datetime import datetime # Importing specific items
  • Use 'import' to include modules
  • Standard library offers many useful modules
  • You can also create your own modules
